# 01 - Fundamentos de SQL

## Objetivo

Entender o que é um banco relacional e como consultar dados com SQL.

## Conceitos

- Banco de dados: local onde dados são armazenados.
- Tabela: conjunto de linhas e colunas.
- Linha: um registro.
- Coluna: um atributo.
- SQL: linguagem usada para consultar e modificar dados.

## Primeiro exemplo

```sql
SELECT nome, idade
FROM alunos;
```

## Missão

Explique com suas palavras:

- o que é uma tabela;
- o que é uma linha;
- o que é uma coluna;
- por que SQL é útil.
