# 06 - Caminho Futuro: DBA, IA e Automação

## Objetivo

Separar o que é SQL de estudo do que será uma trilha futura de administração de banco de dados.

## Temas de DBA para o futuro

- usuários e permissões;
- backup e restore;
- índices;
- performance;
- plano de execução;
- normalização;
- monitoramento;
- logs;
- segurança;
- rotinas automatizadas;
- alta disponibilidade.

## Onde a IA pode ajudar

- explicar consultas;
- sugerir índices;
- revisar modelagem;
- gerar dados de teste;
- criar documentação;
- resumir logs;
- sugerir hipóteses de performance.

## Cuidado

IA não deve executar comandos administrativos sem revisão humana. Em banco de dados, um comando errado pode apagar ou expor informação.
