# 02 - SELECT e Filtros

## Objetivo

Aprender a selecionar colunas e filtrar linhas.

## Comandos

```sql
SELECT *
FROM alunos;

SELECT nome, trilha
FROM alunos;

SELECT nome, pontos
FROM alunos
WHERE pontos >= 50;
```

## Operadores úteis

- `=`
- `<>`
- `>`
- `<`
- `>=`
- `<=`
- `AND`
- `OR`
- `LIKE`
- `IN`

## Missão

Crie consultas para encontrar:

- alunos da trilha Python;
- alunos com mais de 50 pontos;
- alunos de uma cidade específica.
