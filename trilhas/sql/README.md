# Trilha SQL

Uma trilha separada para aprender SQL do básico ao intermediário, preparando caminho para administração de banco de dados no futuro.

## Objetivo

Ensinar o aluno a pensar em dados relacionais:

- tabelas;
- colunas;
- tipos;
- chaves;
- consultas;
- filtros;
- joins;
- agregações;
- modelagem simples;
- boas práticas.

## Para quem é

- Quem já começou Python e quer aprender banco de dados.
- Quem quer entender como dados são armazenados e consultados.
- Quem no futuro quer estudar DBA, performance, segurança e automação.
