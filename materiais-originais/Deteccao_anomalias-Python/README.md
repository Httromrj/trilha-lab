# Detecção de Anomalias em Transações Financeiras

Projeto desenvolvido para detectar fraudes em transações financeiras utilizando Machine Learning em Python.

## Tecnologias

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- SHAP
- Matplotlib

## Etapas do projeto

- Análise exploratória
- Feature Engineering
- Treinamento dos modelos
- Avaliação com métricas
- Balanceamento de dados
- Explicabilidade com SHAP

## Modelos utilizados

- Logistic Regression
- Random Forest
- XGBoost
