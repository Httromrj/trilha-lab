# Projeto 02 - Explorador de Dados

## Objetivo

Ler um CSV e gerar um resumo útil para análise inicial.

## Rodar

```powershell
python src\explorador_dados.py
```

## Melhorias sugeridas

- Receber o caminho do CSV por argumento.
- Salvar relatório em Markdown.
- Gerar gráficos em `outputs/`.
- Detectar colunas categóricas e numéricas automaticamente.
