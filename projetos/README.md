# Projetos

Cada projeto é um laboratório pequeno. Eles não precisam ser perfeitos; precisam ensinar uma ideia.

## Labs

- `01-quiz-python`: interação no terminal e revisão de conceitos.

- `03-mini-banco`: SQLite e persistência.
- `04-primeiro-ml`: regressão linear simples.
- `05-assistente-ia`: desenho do projeto final inspirado pelo TaxPilot.
