# Projeto 03 - Mini Banco SQLite

## Objetivo

Registrar missões de estudo em um banco local.

## Rodar

```powershell
python src\mini_banco.py
```

## Melhorias sugeridas

- Criar menu para cadastrar novas missões.
- Atualizar status pelo terminal.
- Mostrar pontuação total concluída.
- Exportar dados para CSV.
