# 00 - Boas-vindas e Setup

## Objetivo

Preparar o ambiente e entender como a trilha funciona.

## Você vai praticar

- Criar e ativar ambiente virtual.
- Instalar dependências.
- Rodar scripts pelo terminal.
- Registrar aprendizados.

## Missão

Rode o quiz inicial:

```powershell
python src\quiz_trilha.py
```

Depois, crie um arquivo `DIARIO.md` e escreva três linhas:

```text
Hoje eu comecei a trilha.
Meu objetivo é:
Minha maior dúvida agora é:
```

## Conceito-chave

Programar é um ciclo: tentar, observar, corrigir e explicar. O código final importa, mas o raciocínio por trás dele importa mais.
