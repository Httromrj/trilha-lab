# 09 - Projeto Final

## Objetivo

Construir um assistente educacional que organiza dados de uma pessoa fictícia e gera um resumo inteligente.

## Inspiração

O TaxPilot mostra uma boa ideia de produto: guiar o usuário, coletar dados e devolver orientação organizada. Aqui a versão será educacional, sem prometer orientação fiscal ou jurídica.

## Funcionalidades sugeridas

- Cadastro simples de perfil.
- Leitura de dados em CSV ou JSON.
- Validação de campos obrigatórios.
- Armazenamento em SQLite.
- Relatório com pandas.
- Um prompt estruturado para um assistente de IA.
- Métricas simples de qualidade da resposta.

## Entregáveis

- `README.md` do projeto final.
- Script principal executável.
- Dados de exemplo.
- Diagrama simples do fluxo.
- Lista de limitações.
- Ideias de evolução.

## Critérios de sucesso

- O projeto roda localmente.
- A explicação é clara para iniciantes.
- Os dados são tratados com cuidado.
- A IA é usada com contexto e limites.
- O projeto mostra evolução real da trilha.
