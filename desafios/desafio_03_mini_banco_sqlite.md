# Desafio 03 - Mini Banco SQLite

## Nível

Intermediário.

## Missão

Crie um banco SQLite para registrar missões de estudo.

## Requisitos

- Criar tabela `missoes`.
- Inserir pelo menos cinco registros.
- Atualizar uma missão para `concluida`.
- Consultar missões pendentes.
- Consultar pontuação total concluída.

## Campos sugeridos

- `id`
- `titulo`
- `modulo`
- `status`
- `pontos`

## Extra

Crie uma função para listar progresso por módulo.
