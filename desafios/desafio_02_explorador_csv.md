# Desafio 02 - Explorador CSV

## Nível

Iniciante/intermediário.

## Missão

Leia um arquivo CSV e gere um resumo automático.

## Base sugerida

Use `dados/relatorio_gorjetas.csv`.

## Requisitos

- Carregar o CSV com pandas.
- Mostrar quantidade de linhas e colunas.
- Mostrar nomes das colunas.
- Mostrar valores ausentes.
- Calcular estatísticas das colunas numéricas.
- Responder pelo menos uma pergunta de negócio.

## Perguntas possíveis

- Qual dia tem maior média de gorjeta?
- Existe relação entre conta alta e gorjeta alta?
- Qual período teve maior valor médio?

## Extra

Gere um gráfico e salve em uma pasta `outputs/`.
