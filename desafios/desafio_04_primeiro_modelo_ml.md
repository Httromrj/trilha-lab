# Desafio 04 - Primeiro Modelo de ML

## Nível

Intermediário.

## Missão

Treine um modelo simples para prever gorjeta a partir do valor total da conta.

## Requisitos

- Carregar `dados/relatorio_gorjetas.csv`.
- Separar `X` com `total_bill`.
- Separar `y` com `tip`.
- Dividir treino e teste.
- Treinar `LinearRegression`.
- Calcular MAE e R2.
- Fazer uma previsão manual.

## Perguntas para responder

- O modelo parece bom?
- Que variáveis poderiam melhorar a previsão?
- Qual é o risco de confiar cegamente nesse modelo?

## Extra

Compare o modelo usando mais uma variável, como tamanho da mesa.
