# Mídia, Animações e Automações

Esta trilha deve parecer viva: visual, prática e com pequenas recompensas ao longo do caminho.

## Ideias de mídia

| Tipo | Onde usar | Ideia |
| --- | --- | --- |
| Imagem de capa | README e GitHub social preview | Mapa visual da jornada: Python, dados, banco e IA |
| GIF curto | Módulo 00 | Terminal rodando o quiz inicial |
| GIF curto | Lab de execução | Aluno alterando código e vendo o resultado |
| Vídeo de 30s | README | Apresentação rápida da trilha e do projeto final |
| Animação web | Site | Linha de progresso dos módulos |
| Cards visuais | Redes sociais | Um card por módulo com desafio da semana |

## Roteiro de vídeo curto

```text
Cena 1: "Comecei com exemplos soltos de Python."
Cena 2: "Organizei tudo em uma trilha progressiva."
Cena 3: "Agora cada módulo tem desafio, dados e mini projeto."
Cena 4: "No final, um assistente de IA inspirado por problemas reais."
Cena 5: "Projeto: Trilha Lab: Python, SQL, IA, Machine Learning e Dados."
```

## Automações sugeridas

- Validar scripts Python no GitHub Actions.
- Publicar o site estático com GitHub Pages.
- Abrir uma issue automática para cada novo módulo.
- Gerar checklist de revisão antes de publicar uma aula.
- Criar release quando a trilha ganhar um novo bloco completo.

## Experiência divertida

- Pontuação por missão.
- Badges por módulo concluído.
- Quiz no terminal.
- Campos de execução para testar código durante a aula.
- Página visual com trilha animada.
- Desafios pequenos com "extra" para quem quiser ir além.
