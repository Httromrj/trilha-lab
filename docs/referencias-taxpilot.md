# Referência Conceitual: TaxPilot

O TaxPilot é usado aqui apenas como inspiração. Este projeto não altera, copia ou depende do repositório original.

## Ideias reaproveitáveis

- Um assistente precisa guiar o usuário por etapas claras.
- Dados soltos precisam virar contexto organizado.
- O sistema deve explicar limites e pedir informações faltantes.
- A experiência deve ser simples para quem não domina tecnologia.
- A documentação é parte do produto.

## Tradução para esta trilha

Em vez de começar criando um assistente completo, a trilha constrói as peças:

1. Python para manipular informações.
2. Coleções para organizar dados.
3. Arquivos para ler e salvar contexto.
4. Pandas para analisar dados tabulares.
5. Banco para persistir informações.
6. ML para detectar padrões.
7. Prompts para orientar uma conversa.

## Projeto final inspirado

O projeto final será um assistente educacional de organização de dados. Ele pode simular perguntas, validar informações e gerar um resumo, sem prometer consultoria fiscal, financeira ou jurídica.
